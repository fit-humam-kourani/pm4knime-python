This package implements process discovery based on directly-follows graph. The node DFM Miner discovers a directly-follows graph from an event log. The node DFM2PT Miner mines one process tree from directly-follows graph by Inductive Miner.

The DFMMinerNodeModel discovers a directly-follows graph from an event log.

The DFM2PMNodeModel mines one process tree from directly-follows graph by Inductive Miner.

ProM plugin Inductive Miner is used implicitly to get the model directly-follows graph and Proces tree.
